# Atyimo - Record Linkage Application for Heterogeneous Platforms

# Requirements: 
Java 8+
Spark 2.1.x
gcc

# Setup steps:
1. Edit the config.py file
2. Use the 'run_all.sh' script to execute the atyimo.
